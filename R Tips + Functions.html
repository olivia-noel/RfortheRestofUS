mutate() tidyverse

makes or changes existing vars:

  • to make var w/ specific value: mutate(var = "")

  • to make a new var based on other vars: mutate(var1 = var * 2)

  • to change an existing var: mutate(var = var + 1)

    • to change var type: mutate(var = as.numeric(var)) mutate(var = as.character(var))
    • to remove a string of text: mutate(var = str_remove(var,pattern = "text to remove"))
    • to recode specified values: mutate(var = recode(var,"old value" = "new value"))
    • to modify value based on a condition: mutate(var = if_else(var == "some_value",true = "if_true_value",false = "else_value"))
    • to modify value based on multiple conditions: mutate(var = case_when(var == "some value" ~ "new value",var == "some B value" ~ "new B value",var == "some C value" ~ "new C value",.default = "value for all others that don't match above"))
    • to remove non-numeric characters and convert var to numeric: mutate(var = parse_number(var))
    • to set value as NA based on condition: mutate(var = na_if(var, "condition"))
    • to change cell value from NA to specified value: mutate(var = replace_na(var, "value to replace"))
    • to wrap text (to be used with plot labels): mutate(var_wrapped = str_wrap(var, width = Xwidth))
    • to reorder a var for a plot: mutate(var = fct_relevel(var, "cell value1", "cell value2", "cell value3")) or mutate(plottedvar = fct_relevel(plottedvar, var_to_order_by))

group_by() tidyverse

groups data by a specified variable, often used with summarize()

group_by(varX) |>summarize(mean_varY = mean(varY, na.rm = TRUE))

 Week 2: group_by()
summarize() tidyverse

summarizes a variable based on specified argument, for example: summarize(mean_var = mean(var))

must remove NA values using drop_na(var) or in argument: summarize(mean_var = mean(var, na.rm = TRUE))

can have multiple summaries in one function: summarize(mean_var = mean(var, na.rm = TRUE),max_var = max(var, na.rm = TRUE))

can be used with group_by() or can add argument within summarize: summarize(mean_var = mean(var, na.rm = TRUE),.by = c(var1, var2))

geom_text() tidyverse

adds data labels, where label var has been set as argument in ggplot()

possible arguments:

  • vjust = 0 - puts label on outer edge of vertical bars
  • vjust = 1 - puts label on inner edge of vertical bars
  • hjust - used for horizontal justification
  • position=position_stack(vjust=0.5)) - places label in center of bar
  • color = HEX - changes text color
  • fill = HEX - changes background color
  • size - sets label size in mm; can use size = X / .pt) to set size relative to points
  • fontface - font settings, like “bold”
